查询语法

支持的逻辑连接符：&&（与）、||（或）；如果查询表达式有多个与或关系，尽量在外面用（）包含起来；
语法 例句 用途说明 == != >=和 <=
ip ip=="1.1.1.1" 通过单一IPv4地址进行查询 ✓ ✓ -
ip=="220.181.111.1/24" 通过IPv4掩码进行查询 ✓ ✓ -
port port=="6379" 通过端口号进行查询 ✓ ✓ -
domain domain=="qq.com" 通过根域名进行查询 ✓ ✓ -
host host=="api.fofa.info" 通过主机名进行查询 ✓ ✓ -
location location=="北京" 通过IP归属地进行查询 ✓ ✓ -
status status=="200" 通过HTTP状态码进行查询 ✓ ✓ -
service service=="http" 通过服务进行查询 ✓ ✓ -
banner banner=="ssh" 通过协议返回信息进行查询 ✓ ✓ -
server server=="Microsoft-IIS/10" 通过服务器进行查询 ✓ ✓ -
cert cert=="fofa.info" 通过TLS证书进行查询 ✓ ✓ -
icon_hash icon_hash=="-247388890" 通过网站图标的hash值进行查询 ✓ ✓ -
title title=="beijing" 通过网站标题进行查询 ✓ ✓ -
header header=="elastic" 通过响应标头进行查询 ✓ ✓ -
body body=="网络空间测绘" 通过HTML正文进行查询 ✓ ✓ -
org org=="测试单位" 通过所属的Nemo自定义的组织机构查询 ✓ ✓ -
create_time create_time=="2024-07-01" 通过资产的创建时间查询（日期格式为yyyy-MM-dd） ✓ ✓ ✓
update_time update_time=="2024-07-01" 通过资产的最后更新时间查询（日期格式为yyyy-MM-dd） ✓ ✓ ✓
